TROUBLESHOOTING QUICK REFERENCE

Problem: "Cannot connect to MongoDB"
Solution: Check .env file, verify MongoDB URI, check IP whitelist

Problem: "CORS Error" in browser
Solution: Ensure backend running on :5000, check DevTools Network

Problem: "Socket.IO connection failed"
Solution: Backend must be running, check port 5000 access

Problem: "Port already in use"
Solution: lsof -ti:PORT | xargs kill -9

Problem: "Module not found"
Solution: Run npm install in that directory

Problem: Don't know what to do
Solution: Read START_HERE.txt
