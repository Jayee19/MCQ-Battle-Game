MCQ 1vs1 BATTLE GAME - PROJECT SUMMARY
=====================================

PROJECT OVERVIEW
================
Full-stack real-time multiplayer quiz application where users compete in MCQ-based challenges.

TECH STACK
==========
Frontend: React 18, Bootstrap 5, Socket.IO, Chart.js, Axios
Backend: Node.js/Express, MongoDB, Mongoose, JWT, Bcrypt
Real-time: Socket.IO, Pusher

KEY FEATURES
============
✓ Real-time multiplayer gameplay
✓ User authentication (JWT)
✓ Game rooms & matchmaking
✓ Live scoring
✓ Dashboard with analytics
✓ Leaderboard system
✓ Admin panel for quiz management

QUICK START
===========
1. Clone: git clone https://github.com/yogeshlakwal3114/MCQ-Battle-Game.git
2. Backend setup:
   - cd Backend
   - npm install
   - Create .env file with MONGODB_URI, JWT_SECRET, PORT
   - npm run dev

3. Frontend setup:
   - cd Frontend
   - npm install
   - npm start

TESTING

Manual tests to perform:

TROUBLESHOOTING
===============
MongoDB Connection: Check .env file and MongoDB Atlas IP whitelist
CORS Error: Verify both backend and frontend are running
Socket.IO Issues: Check ports and firewall settings
Port Already in Use: Use lsof -ti:PORT | xargs kill -9
